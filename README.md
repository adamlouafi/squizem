# Squizem
## About
Squizem is a simple quiz app developed with [Vue 3](https://vuejs.org/guide/introduction.html) and [Tailwind CSS](https://tailwindcss.com/docs/installation) and using the [Open Trivia Database](https://opentdb.com/).
